Assumptions & reasoning
Constant across all four directions (locked by the PRD and your answers):
• Type trio: Newsreader for headings, Source Sans 3 for body, IBM Plex Mono for tokens, code and cell tags.
• One accent is reserved exclusively for the sink: token-0 chip, first heatmap column, headline metric, sink curves. Nothing else gets it.
• Real tokenization shown (the · marks a leading space); causal mask drawn as near-empty cells; query row 0 excluded from the headline metric — per PRD §11.
Hybrid marimo affordance — per your pick: no heavy notebook chrome, but every panel carries a small mono cell tag (mo.ui.slider, altair…) naming the marimo element it maps to, so Claude Code can translate panel-for-panel.
What varies here: theme (2 light / 2 dark), accent hue, heatmap colormap, and hero layout density — from editorial two-column to dense console.
